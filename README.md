# Friendrive Client

## Introduce

가족, 지인, 방문 연수 등 비전문적으로 운전을 배우는 미숙한 면허 취득자 들이 커리큘럼, 가이드,
진척도 등의 정보를 강사에게 공유하고, 연수 과정을 스스로 관리하며 주도하는 운전 가이드 서비스
